Telegram Bot

⚠️ Не забудь вставить токен бота, получить его можно в @botfather

Инструкция по созданию файла

1. Установить pyinstaller:

pip install pyinstaller

2. Создать файл

pyinstaller --noconsole --onefile ваш_файл.py

Готовый файл будет в папке dist
